Cadastre Vector Tiles

Production release 2022-06-01 as Version v3

  1. Get The free API Key - and get access to Free of Charge Services
  2. Fork one of the examples or download a StyleJSON Template using links below
  3. Edit the style to make a custom map for your application
  • Map Demo with OpenLayers (7.1.0) (Source code at GitHub)
  • Documentation for Cadastre Vector Tiles (in finnish)
  • Cadastre Vector Tiles are available with the API-key
  • Cadastre Vector Tiles contain both register data properties and geometries.
  • Cartographic generalization is applied to geometries on some tilematrix levels
  • Default projection is WebMercator. ETRS-TM35FIN is supported with ?TileMatrixSet=ETRS-TM35FIN
  • Cadastre boundaries StyleJSON template
    (compatible with QGIS)
  • Cadastre boundaries StyleJSON template with some property based visualisation
    (compatible with QGIS)
  • Quick Start for QGIS
  • Map Demo with Maplibre GL JS
    • Source Code at GitHub
  • Documentation for Cadastre OGC API features (in finnish)
  • Cadastre OGC API Features requires registration
  • Cadastre OGC API Features contain both register data properties and geometries.
  • No cartographic generalization is applied to geometries

Topographic Vector Tiles

Production release 2020-06-02 as Version v20

  1. Get The free API Key - and get access to Free of Charge Services
  2. Download the Generic StyleJSON Template using links below
  3. Edit the style to make a custom map for your application
  • Demo with Style and language selection and Geocoding
  • Documentation for Topographic Vector Tiles (in finnish)
  • Topographic Vector Tiles are available with the API-key
  • Topographic Vector Tiles contain both register data properties and geometries.
  • Cartographic generalization is applied to geometries on some tilematrix levels
  • Default projection is WebMercator. ETRS-TM35FIN is supported with ?TileMatrixSet=ETRS-TM35FIN
  • Generic StyleJSON Template for WebMercator
    This style implements only a raw visualization for some layers
  • StyleJSON Template for WebMercator
    Contains rules for roads, tunnels, bridges, junctions
  • Vector Tiles Map with Style selection and Geocoding support
    Four different styles and language selection of finnish, swedish and sami languages
  • Custom styled Map with contour lines, scarps and other terrain
  • Background Map (ETRS-TM35FIN)
  • Hybrid Vector with Orthophoto Map (ETRS-TM35FIN)
  • Generic Style Map (ETRS-TM35FIN)
  • Custom styled Map with contour lines, scarps and other terrain (ETRS-TM35FIN)
  • Codelist Catalogue for Vector Tiles properties
  • Codelist Catalogue for Geocoding Places properties

Client application and library compatilibity

  • Vector Tiles are known to be compatible with
    Maputnik, Maplibre GL JS, Maplibre Native (Android), QGIS,
    Mapbox GL JS|Mapbox (v1), Maps SDK for Android|Mapbox (v1), Maps SDK for Qt 5.15|Mapbox (v1)    ,
    OpenLayers, Oskari, Suomi.fi-kartat
  • The OGC API Simple Features service endpoint is known to be compatible with GDAL, QGIS, ArcGIS Online and with any JSON or GeoJSON compatible library (such as OpenLayers)
